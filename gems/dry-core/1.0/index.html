Introduction

dry-core is a simple toolset that can be used in many places.

Features

  • Cache - allows you to cache call results that are solely determined by arguments.
  • Class Attributes
  • Class Builder
  • Constants - a list of constants you can use to avoid memory allocations or identity checks.
  • Deprecations
  • Equalizer - simple mixin providing equality, equivalence and inspection methods.
  • Extensions
